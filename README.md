## 🦊 Introducción
En esta práctica se desarrolla el modelado de un sistema electrico, utilizando herramientas de simulación como MATLAB, Simulink y Simscape.
---

🍁🍂🍁🍂🍁🍂🍁🍂🍁🍂🍁🍂🍁🍂🍁🍂

## 🍂 Resultados

Las simulaciones realizadas permiten observar el comportamiento dinámico del sistema mecánico modelado.

Entre los principales resultados obtenidos se encuentran:

* Evolución de la **posición del sistema**
* Variación de la **velocidad a lo largo del tiempo**
* **Respuesta temporal** generada por el modelo matemático
